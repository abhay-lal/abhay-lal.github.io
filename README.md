# Abhay Lal - Portfolio

Modern React-powered portfolio website with smooth animations built using Framer Motion.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start
```

Visit `http://localhost:3000` to see your portfolio!

## 🌐 Deploy

```bash
# Deploy to GitHub Pages
npm run deploy
```

## 📁 Project Structure

```
src/
├── components/          # React components
│   ├── Header.js       # Navigation
│   ├── Home.js         # Hero section
│   ├── About.js        # About section
│   ├── Skills.js       # Skills with accordion
│   ├── Experience.js   # Timeline
│   ├── Projects.js     # Projects carousel
│   ├── Publications.js # Publications carousel
│   ├── Contact.js      # Contact info
│   ├── Footer.js       # Footer
│   └── ScrollUp.js     # Scroll button
├── App.js              # Main app
├── index.js            # Entry point
└── index.css           # Global styles
```

## 🎨 Customization

### Change Color Scheme

Edit `src/index.css`:
```css
:root {
  --hue-color: 230; /* Blue (default) */
  /* Try: 142 (Green), 250 (Purple), 340 (Pink) */
}
```

### Update Content

Edit component files in `src/components/` to update your content.

## 🛠️ Technologies

- **Framer Motion** - Animations
- **Swiper** - Carousels
- **React Intersection Observer** - Scroll animations
